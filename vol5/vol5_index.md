# 卷五：专项篇

## 本卷简介

专项篇聚焦 Agent 在垂直领域的深度实践。当基础架构与高级能力已经就位，真正释放价值的是将 Agent 嵌入具体工作场景——写代码、分析数据、创作内容、编排工作流。本卷逐一拆解四大核心场景，从原理到生产级实现，提供可直接落地的方案。

## 章节目录

| 章节 | 标题 | 核心主题 | 难度 |
|------|------|----------|------|
| 第16章 | 代码 Agent | 代码生成、审查、重构、Debug、项目脚手架 | ★★★★☆ |
| 第17章 | 数据分析 Agent | 自然语言查询、数据清洗、统计可视化、报告生成 | ★★★★☆ |
| 第18章 | 内容创作 Agent | 文章写作、营销文案、多语言翻译、SEO优化 | ★★★☆☆ |
| 第19章 | 自动化工作流 Agent | 工作流引擎、条件分支、人工审批、事件触发 | ★★★★★ |

## 阅读建议

- **开发者**：优先阅读第16章，将 Agent 融入日常开发流程
- **数据从业者**：第17章是必读，Text-to-SQL 与自动化报告是高频需求
- **运营与市场人员**：第18章提供系统化的内容创作方法论
- **架构师**：第19章的工作流设计模式对构建复杂 Agent 系统至关重要

## 本卷依赖

- 卷一（认知篇）：Agent 的基本概念与设计原则
- 卷二（基础篇）：Tool Use、Prompt Engineering、RAG 基础
- 卷三（进阶篇）：多 Agent 协作、记忆系统、安全防护
- 卷四（高级篇）：可观测性、评测体系、生产部署

建议按顺序阅读，如已有相关经验可按需跳转。
