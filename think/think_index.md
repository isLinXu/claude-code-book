# 特别篇：Agent 编程思想与前沿模型

> **两个维度解读 Agent 开发的深层逻辑：**
> - 🔬 **编程思想**（ch01~ch09）：以 Claude Code v2.1.88 源码为标本，解剖 Agent 式编程的设计哲学
> - 🧬 **前沿模型**（ch10~ch11）：Claude Opus 4.6 System Card 精华，213 页技术报告中的能力、安全与对齐

---

## 为什么需要这一篇？

正文 12 卷 48 章讲的是 **"如何构建 Agent"**——概念、技术、模式、工程。

本篇补充两个更深的维度：

1. **Agent 是如何思考的？** 透过一个顶级 Agent 产品的 16,668 行源码，理解编程范式从指令式 → 对象式 → Agent 式的根本性转变
2. **前沿模型能做什么？** 通过 Opus 4.6 的 213 页 System Card，理解当前最强 Agent 基座模型的能力边界和安全约束
